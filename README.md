# Клинический гайд: Показания к витреоретинальной хирургии

Интерактивный веб-справочник для врачей-офтальмологов поликлинического звена.

## Содержание

- Обязательный протокол обследования
- Показания по 13 нозологиям с ОКТ-критериями
- Уровни срочности направления (экстренно / срочно / планово)
- Сводная таблица быстрого принятия решений
- Красные флаги
- Глоссарий сокращений

## Использование

Откройте `index.html` в браузере — без сервера, без зависимостей.

Или посмотрите на GitHub Pages: https://zrachokodintsov2026.github.io/vitreorenal-guide/
