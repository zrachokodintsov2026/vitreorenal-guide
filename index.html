Раздел 2. Показания по нозологиям

🔴 ЭКСТРЕННО

2.1 Отслойка сетчатки

Регматогенная / Тракционная / Экссудативная

Жалобы — НАПРАВЛЯТЬ НЕМЕДЛЕННО:

  • «Шторка», «занавеска», «тень» в поле зрения (любой локализации)
  • Внезапное снижение остроты зрения
  • Вспышки света, фотопсии — предшественники разрыва
  • «Дождь из сажи», «туман» — нередко гемофтальм при разрыве

Данные обследований:

МетодКритерии направления
ВизометрияЛюбое снижение МКОЗ + клиника отслойки
ОфтальмоскопияКуполообразное выбухание серовато-белого цвета, складки сетчатки, видимые разрывы
ОКТСубретинальная жидкость (СРЖ), расслоение нейросенсорного слоя
B-сканированиеПодвижная эхогенная мембрана. При гемофтальме — единственный метод
ТонометрияГипотония < 8 мм рт.ст. — косвенный признак отслойки
ПериметрияАбсолютный дефект, соответствующий зоне отслойки
✅ ВСЕ регматогенные отслойки сетчатки подлежат хирургическому лечению. Выбор метода — решение хирурга.
🟡 ПЛАНОВО-СРОЧНО

2.2 Макулярное отверстие (МО)

Жалобы:

  • Выпадение или искажение центральной части поля зрения
  • Метаморфопсии — тест Амслера положительный
  • Снижение МКОЗ от умеренного до значительного
  • Трудности при чтении, работе с мелкими предметами

Стадии МО (по Gass):

СтадияОКТ-признакиРешение
1А (ВМТ)ПВО частичное, интраретинальные кисты, без сквозного отверстияНаблюдение или направление
1БПсевдоотверстие или ламеллярное отверстиеОКТ каждые 3 мес; при прогрессии — направить
2 — НАПРАВИТЬСквозное МО < 400 мкм, частичное ПВО≤ 4 недели
3 — НАПРАВИТЬМО > 400 мкм, полное ПВО≤ 4 недели
4 — НАПРАВИТЬМО > 400 мкм + опекулум, МКОЗ < 0,2Направить

Ключевые ОКТ-параметры для направления:

  • Минимальный диаметр МО (мкм) — горизонтальный скан
  • Базальный диаметр МО (мкм)
  • Индекс МО (MI = высота/ширина) — при MI < 0,5 прогноз хуже
  • Наличие/отсутствие ЭРМ
  • Состояние эллипсоидной зоны (IS/OS) вокруг отверстия
🟢 ПЛАНОВО

2.3 Эпиретинальная мембрана (ЭРМ)

Жалобы:

  • Метаморфопсии (искажения) — основная жалоба
  • Снижение МКОЗ (вариабельно)
  • Монокулярная диплопия, микропсия

Наблюдать vs Направить:

НаблюдатьНаправить к хирургу
МКОЗ ≥ 0,7, жалоб нет или минимальныМКОЗ ≤ 0,5 с жалобами на метаморфопсии
Тонкая ЭРМ без деформации слоёвВыраженные метаморфопсии при любой МКОЗ
Стабильная картина при контрольной ОКТПрогрессирование по ОКТ: нарастание отёка, кист
Пациент без функциональных жалобУтолщение сетчатки > 400 мкм в фовеа
—ЭРМ + подозрение на ламеллярное/сквозное МО
🟢/🟡 ПЛАНОВО

2.4 Витреомакулярная тракция (ВМТ) и адгезия (ВМА)

ОКТ-критерии:

  • ВМА: прикрепление задней гиалоидной мембраны к фовеа < 1500 мкм без деформации → наблюдение
  • ВМТ: прикрепление СТ + деформация/утолщение + кисты → направить
  • Диаметр адгезии > 1500 мкм + деформация → направить
  • Прогрессия ВМА → ВМТ → МО → направить
ℹ️ ВМА без деформации нередко разрешается самостоятельно. Повторная ОКТ через 3 месяца при бессимптомной ВМА.
🟠 СРОЧНО / 🔴 ЭКСТРЕННО

2.6 Тромбоз / Окклюзия ЦВС и ветвей (ОВС)

Жалобы и клиника:

  • Острое/подострое безболезненное снижение зрения
  • «Туман», «пелена» перед глазом
  • При ОВС — дефект поля в одном квадранте/секторе

ОКТ и показания к направлению:

  • ДМО на фоне тромбоза (CST > 300 мкм) → направить для анти-VEGF / ВЭ
  • Резистентный к ИВВ анти-VEGF макулярный отёк (> 6 инъекций) → на ВЭ
  • Неоваскуляризация сетчатки / ДЗН → ПФК или ВЭ
  • Гемофтальм на фоне тромбоза → срочно
  • Тракционная ОС на фоне тромбоза → экстренно
  • Хронический ДМО > 12 месяцев без регресса → рассмотреть ВЭ
ℹ️ Первые 3 месяца: ОКТ ежемесячно. Оценить ДМО, ОКТ-А, ФАГ.
🟠 СРОЧНО (≤ 7 дней)

Форма ВМДОКТ / Клинические признакиТактика

ХНВ + субмакулярное кровоизлияниеТёмно-красный преретинальный очаг в макулеЭКСТРЕННО (≤ 48 ч)

ОКТ-признаки активной влажной ВМД — ОБЯЗАТЕЛЬНОЕ НАПРАВЛЕНИЕ:

  • ИРЖ — гипорефлективные кисты внутри слоёв сетчатки
  • СРЖ — жидкость между ПЭС и нейросенсорным слоем
  • ОПЭ — куполообразное возвышение ПЭС
  • СГРМ — мембрана ХНВ
  • Double layer sign — ХНВ 1 типа
⚠️ При любом впервые выявленном случае влажной ВМД — направить к ретинологу в течение 7 дней! Потеря фоторецепторов необратима!
🟠 СРОЧНО

2.8 Гемофтальм

Причины, требующие срочного направления:

  • Пролиферативная диабетическая ретинопатия (наиболее частая)
  • ЗОСТ с разрывом сосуда / сетчатки
  • Тромбоз ЦВС / ветвей
  • Отслойка сетчатки с кровью
  • Травматический гемофтальм

Обследование и показания:

  • МКОЗ < 0,1 + помутнение СТ → B-сканирование обязательно
  • Гемофтальм без видимой причины у молодых → исключить разрыв сетчатки
  • Рецидивирующий гемофтальм → хирургия (витрэктомия)
  • Срок резорбции > 3 месяцев без улучшения → витрэктомия
⚠️ B-сканирование обязательно при гемофтальме! Отслойка сетчатки под кровью = ЭКСТРЕННОЕ направление!
🟠 СРОЧНО

2.9 Пролиферативная витреоретинопатия (ПВР)

  • Тяжи и мембраны в СТ и на поверхности сетчатки → витрэктомия
  • Фиксированные складки сетчатки (ПВР С или Д) → срочно
  • Рецидив отслойки сетчатки → направить повторно
  • Укороченная сетчатка, не поддающаяся расправлению → хирургия с силиконом
🟡 ПЛАНОВО-СРОЧНО

2.10 Хориоидальная неоваскуляризация (не-ВМД)

Причины ХНВ, не связанные с ВМД:

  • Миопическая ХНВ (патологическая миопия, стафилома)
  • Ангиоидные полосы
  • Хориоидит, хориоретинальные рубцы
  • Идиопатическая ХНВ у молодых

ОКТ-показания к направлению:

  • Серо-зелёный субретинальный очаг + СРЖ или ИРЖ → направить
  • ОПЭ + СГРМ → направить
  • Снижение МКОЗ + подозрение на ХНВ → ОКТ-А или ФАГ → направить
  • Миопическая ХНВ: любой размер при миопии > -6 дптр + жалобы → направить
🟢 ПЛАНОВО

2.11 Центральная серозная хориоретинопатия (ЦСХРП)

Жалобы и ОКТ:

  • Острое/подострое снижение МКОЗ, микропсия, хроматопсия
  • ОКТ: куполообразная СРЖ под нейросенсорной сетчаткой ± отслойка ПЭС
  • Хроническая ЦСХРП (> 3–4 месяцев) → ФДТ или термическая лазерная коагуляция
  • Прогрессирующая атрофия ПЭС, нарастание СРЖ → направить
  • МКОЗ < 0,5 на фоне ЦСХРП → направить
ℹ️ Острая ЦСХРП (< 3 мес): нередко самостоятельное разрешение. Исключить кортикостероиды! Контрольная ОКТ через 6–8 недель.
🔴 ЭКСТРЕННО

2.12 Травма глаза

⚠️ ЛЮБАЯ проникающая травма → экстренное направление в специализированный стационар!
  • Тупая травма + снижение зрения → B-скан, ОКТ → при гемофтальме / ОС → срочно
  • Макулярное отверстие травматическое → направить к ВР-хирургу
  • Хориоидальный разрыв через фовеа → направить (риск ХНВ)
  • Субретинальное кровоизлияние → направить (удаление, фармакологический лизис)
🔴 ЭКСТРЕННО

2.13 Ретинопатия недоношенных (РН)

  • 3+ стадия и выше → срочное направление
  • «Rush disease» (быстро прогрессирующая задняя РН) → экстренно
  • Стадия 4А (экстрафовеальная тракционная ОС) → витрэктомия
  • Стадия 4Б и 5 → витрэктомия (прогноз зрения хуже при промедлении)

Раздел 5. «Красные флаги» — немедленное направление

1
«Шторка» / «завеса» в любом секторе поля зрения — отслойка сетчатки до распространения на макулу
2
Острая потеря зрения на фоне субмакулярного кровоизлияния — первые 24–48 часов решают прогноз
3
Проникающая травма глаза — любая степень тяжести
4
Рубеоз радужки у пациента с ПДР/тромбозом → угроза неоваскулярной глаукомы
5
Тракционная отслойка сетчатки при ПДР — любой локализации
6
Ребёнок недоношенный: быстро прогрессирующая задняя РН (Rush disease)
